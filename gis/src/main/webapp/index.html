Beispiele

Folgende Beispiele stehen zur Verfügung:

  1. Statische Daten anhand von Spielplätzen in Köln
  2. Statische Daten anhand von Kioske in NRW;
    Datenextrakt aus Open Streetmap Daten,
    unterschiedliche Darstellungsmöglichkeiten von Punkten
  3. Per Ajax/ REST abgerufene Shop-Informationen
  4. Interaktiv administrative Layer aus OSM abfragen und anzeigen
  5. Per Ajax/ REST abgerufene Autbahn-Informationen
  6. Interaktiv Kioske abfragen, anzeigen und suchen
  7. Zeichnen von Layern - in der Entwicklung

Browsersupport

Im Rahmen dieses Projektes wird für die grundlegenden Funktionen neben Leaflet jQuery eingesetzt. Da es in diesem Projekt nicht um die Darstellung der größtmöglichen Kompatibilität geht, wird die aktuellste Version (jQuery 2.1.1) eingesetzt. Das bringt es mit sich, dass der Internet Explorer erst ab Version 9 unterstützt wird. Die generelle Unterstützung der Browser befindet sich auf den jQuery-Seiten im Bereich Browser Support.

Besonderheiten

Dieser Showcase soll die Möglichkeiten eines WebGis aufzeigen. Es ist aber auch der Ort, um Lernziele darzustellen.

Um die Datenbasis überschaubar zu halten, liegen die OpenStreetMap Daten für das Land Nordrhein-Westfalen zugrunden.
